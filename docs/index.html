    ncurses

    Ncurses Bindings for Crystal

    Installation

    Add this to your application's shard.yml:

    dependencies:
  ncurses:
    github: SamualLB/ncurses

    Usage

    require "ncurses"

    Basic Printing ####

    NCurses.init

NCurses.stdscr.print "Hello world!"

NCurses.end_win
